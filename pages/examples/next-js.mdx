# Install HappyPath with Next.js

How to integrate HappyPath into a Next.js app.

```js filename="happypath.config.js"
import getClient from '@happypath-io/remote-config'

//load config from API on happypath.io
const remoteConfig = getClient({
  apiKey: '123456',
})
```

```js filename="__app.tsx"
await remoteConfig.init()
```
