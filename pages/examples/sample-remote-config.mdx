# Sample Remote Config

Here is an example of what a remote configuration file looks like. You can create and host your own, or use [happypath.io](https://app.happypath.io) to create and manage them in a web interface.

<Callout type="info">
  Check out installation instructions [here](/installation)
</Callout>

```json filename="config.json"
{
  "name": "Bob",
  "age": 30,
  "isEngineer": true,
  "roles": ["engineer", "developer"],
  "metadata": {
    "states": ["CA", "MD"],
    "rules": {
      "allowed": false,
      "travelCount": 10
    }
  }
}
```
